# Security Policy

We take security research and disclosure seriously. If you discover a vulnerability
or an issue in code that might be misused, please follow responsible disclosure:

- Do not publicize the issue before contacting the maintainer: voltsparx@gmail.com
- Provide clear repro steps, affected versions, and suggested mitigations.

This repository intentionally contains only benign simulation code. If you find any
code that could be dangerous, report it immediately.
